There are various types of multiscale methods, etc. This list is in no way exhausting. The methods below are in alphabetic order. In the coming years, there will undoubtedly be more applications to which multiscale analysis may lend itself. Please feel free to contact us if you want to add your own multiscale methods.

Concurrent Multiscale Methods

  • Concurrent atomistic-continuum (CAC)
  • Coupled Atomistic Discrete Dislocation (CADD)
  • Embedded Statistical Coupling Method (ESCM)
  • Particle-Based Multiscale Analysis Program (PMAP)
  • Quasi-Continuum (QC)

Hierarchical Multiscale Methods

  • Discrete Dislocation Dynamics (DDD)
  • Multilevel Finite Element Method (FE2)
